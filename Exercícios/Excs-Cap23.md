## Exercise 1

$f(x) = q(x)g(x) + r(x)$

- $f(x) = x^6 + 3x^5 + 4x^2 - 3x + 2$
- $g(x) = x^2 + 2x - 3$
- $q(x) = x^4 + x^3 + x^2 + x - 2$
- $r(x) = 4x + 3$
